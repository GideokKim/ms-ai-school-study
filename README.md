## 🛠️ Setup

1. Create a virtual environment:
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

2. Install required packages:
```bash
pip install numpy matplotlib jupyter scipy
```

3. Launch Jupyter:
```bash
jupyter notebook
```
